# Similar Trello (Kanban Project Management Tool)

## Overview
This project is a Kanban-style project management web application inspired by Trello. It allows users to create boards, manage lists and cards, and organize tasks visually using drag-and-drop functionality.

## Tech Stack
Frontend:
- React (Vite)

Backend:
- Node.js
- Express.js

Database:
- MySQL
- Prisma ORM

## Features

Board Management:
- Create new boards
- View all boards
- Delete boards

List Management:
- Create lists within a board
- Display lists in columns

Card Management:
- Create cards inside lists
- Drag and drop cards between lists
- Reorder cards visually
- Delete cards

User Experience:
- Clean UI with Trello-like layout
- Responsive and simple design
